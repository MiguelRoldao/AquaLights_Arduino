# AquaLights_Arduino

This project aims to create a totally controllable LED light fixture for aquariums. Interfaceable thorugh a 16x2 text LCD and 4 buttons for simple use.

It features 5 colour channels:
- Violet (425nm)
- Royal Blue (445nm)
- Blue (465nm)
- Cold White (6500K)
- Warm White (3500K)
